# 泛微OA漏洞合集

**当前集合漏洞：**

```
泛微云桥任意文件读取

泛微OA V8前台Sql注入

泛微OA WorkflowServiceXml RCE

泛微OA weaver.common.Ctrl 任意文件上传
```

先写了这些，也欢迎补充～

其中`/poc`下的利用脚本均可独立使用。

**Usage: **

```bash
python3 main.py -f filename

python3 main.py -u url
```
